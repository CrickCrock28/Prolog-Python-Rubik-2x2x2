cube_utils.pl -- 2x2x2 Rubik's Cube Utilities

 state_zero(-State:atom)
Provides the solved state of the 2x2x2 Rubik's Cube.
Arguments:
State- The solved state, represented as an atom.
 move(?Move:atom)
Defines valid moves for the 2x2x2 Rubik's Cube.
Arguments:
Move- A valid move (e.g., down_clockwise, back_counterclockwise).
 reversal(+Move:atom, -Reversal:atom)
Provides the reversal of a given move.
Arguments:
Move- The original move.
Reversal- The move that reverses the original move.
 check_color_count(+State:atom, +Color:char, +ExpectedCount:int)
Validates if a specific color appears the expected number of times in the state.
Arguments:
State- The cube state.
Color- The color to validate.
ExpectedCount- The expected count of the color.
 valid_state(+State:atom)
Checks if the given cube state is valid.
Arguments:
State- The cube state to validate.
 apply_path(+Start:atom, +Moves:list)
Applies a series of moves to a cube state
Arguments:
Start- The starting cube state.
Moves- The list of moves to apply.
 shuffle(+Rand:atom)
Generates a randomly shuffled cube state.
Arguments:
Rand- The resulting shuffled state.
 move_transform(+In_state:atom, +Move:atom, -Out_state:atom)
Transforms a cube state based on a move.
Arguments:
In_state- The input cube state.
Move- The move to apply.
Out_state- The resulting cube state after applying the move.
